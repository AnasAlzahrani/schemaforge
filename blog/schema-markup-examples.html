📄 Article Schema Example

Perfect for blog posts, news articles, and editorial content. Article schema can help your content appear in Google's "Top Stories" and other article-focused search features.

JSON-LD Code

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "How to Bake the Perfect Chocolate Chip Cookies",
  "description": "Learn the secret techniques for making soft, chewy chocolate chip cookies that everyone will love.",
  "image": [
    "https://example.com/cookies-1x1.jpg",
    "https://example.com/cookies-4x3.jpg",
    "https://example.com/cookies-16x9.jpg"
  ],
  "author": {
    "@type": "Person",
    "name": "Jane Baker",
    "url": "https://example.com/author/jane-baker"
  },
  "publisher": {
    "@type": "Organization",
    "name": "Baking Today",
    "logo": {
      "@type": "ImageObject",
      "url": "https://example.com/logo.jpg"
    }
  },
  "datePublished": "2026-02-01T08:00:00+00:00",
  "dateModified": "2026-02-01T14:30:00+00:00",
  "mainEntityOfPage": {
    "@type": "WebPage",
    "@id": "https://example.com/chocolate-chip-cookies"
  }
}
</script>

    "addressRegion": "NY",

    "addressCountry": "US"
  },

  "telephone": "+1-212-555-0123",
  "url": "https://bellaitalianyc.com",

  "openingHours": [
    "Mo-Th 11:00-22:00",

🍳 Recipe Schema Example

Great for food blogs and recipe sites. Recipe schema can display cooking time, difficulty, ratings, and even carousel results in search.

JSON-LD Code (Simplified)

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Recipe",
  "name": "Classic Spaghetti Carbonara",
  "description": "Authentic Roman pasta dish with eggs, cheese, and pancetta.",
  "image": "https://example.com/carbonara.jpg",
  "author": {
    "@type": "Person",
    "name": "Chef Marco"
  },
  "prepTime": "PT15M",
  "cookTime": "PT15M",
  "totalTime": "PT30M",
  "recipeYield": "4 servings",
  "recipeCategory": "Dinner",
  "recipeCuisine": "Italian",
  "nutrition": {
    "@type": "NutritionInformation",
    "calories": "520 calories"
  },
  "recipeIngredient": [
    "400g spaghetti",
    "200g pancetta, diced",
    "4 large eggs",
    "100g Pecorino Romano cheese, grated",
    "Black pepper, freshly ground"
  ],
  "recipeInstructions": [
    {
      "@type": "HowToStep",
      "text": "Cook spaghetti in salted boiling water until al dente."
    },
    {
      "@type": "HowToStep", 
      "text": "Fry pancetta until crispy. Beat eggs with cheese and pepper."
    },
    {
      "@type": "HowToStep",
      "text": "Toss hot pasta with pancetta, then with egg mixture off heat."
    }
  ]
}
</script>

❓ FAQ Schema Example

FAQ schema can get your Q&A content featured directly in search results with expandable sections, providing immediate value to searchers.

JSON-LD Code

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "What is schema markup?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "Schema markup is structured data that helps search engines understand your website content better. It can lead to rich snippets in search results."
      }
    },
    {
      "@type": "Question", 
      "name": "Is schema markup required for SEO?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "Schema markup is not required, but it's highly recommended. It can improve your search visibility and click-through rates significantly."
      }
    },
    {
      "@type": "Question",
      "name": "Which schema format should I use?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "JSON-LD is Google's preferred format. It's cleaner than microdata and doesn't interfere with your HTML structure."
      }
    }
  ]
}
</script>

For individual reviews of products, services, or businesses.

For video content that can appear in video search results.

Great for video-rich pages

✨ Schema Implementation Best Practices

✅ Do's

  • Use JSON-LD format - Google's preferred choice
  • Include required properties - Check schema.org documentation
  • Match visible content - Schema should reflect what users see
  • Test before publishing - Use Google's Rich Results Test
  • Be specific - Use the most precise schema type available
  • Update regularly - Keep schema current with content changes

❌ Don'ts

  • Don't markup hidden content - Google may penalize this
  • Don't use generic types - Be as specific as possible
  • Don't ignore errors - Fix validation issues promptly
  • Don't duplicate schema - One schema per content piece
  • Don't stuff keywords - Use natural, accurate descriptions
  • Don't use outdated syntax - Stay current with schema.org updates

🎯 Pro Tip: Start Small, Then Expand

Begin with basic schema for your most important content types (Article, Product, LocalBusiness), then gradually add more specific markup as you get comfortable with the process.
