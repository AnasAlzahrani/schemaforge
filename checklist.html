What's inside

  1. Research question — Is it answerable with available data?
  2. Study type match — Does your design actually answer the question?
  3. Causal structure — DAG drawn? Confounders identified?
  4. Sample size — Power calculation with realistic effect size?
  5. Primary outcome — Single, pre-specified, clinically meaningful?
  6. Comparison group — Active comparator or true control?
  7. Selection bias — Enrollment criteria create systematic differences?
  8. Information bias — Measurement error differential by group?
  9. Time zero — Is follow-up aligned with treatment initiation?
  10. Immortal time — Gap between eligibility and treatment start?
  11. Missing data plan — MCAR/MAR/MNAR classified?
  12. Competing risks — Death before event censored correctly?
  13. Multiple comparisons — Adjustment planned?
  14. Sensitivity analysis — E-value or Rosenbaum bounds?
  15. Analytic code — Reproducible? Version controlled?
  16. Reporting guideline — STROBE / CONSORT / PRISMA selected?
  17. Pre-registration — Protocol registered before analysis?
