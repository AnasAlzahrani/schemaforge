How to Use Event Schema Markup

      

        

          
1

          
Enter Event Info

          
Fill in the event name, start/end dates, and description. Use ISO 8601 date format for best results.

        

        

          
2

          
Add Location

          
Include the venue name and full address. Google uses this to show events in local search.

        

        

          
3

          
Add Ticket Info

          
Include pricing and availability if applicable. This helps users decide directly from search results.

        

        

          
4

          
Implement & Monitor

          
Add the JSON-LD to your event page and check Google Search Console for event appearance data.
