How to Use How-To Schema Markup

      

        

          
1

          
Enter How-To Details

          
Add the title, description, estimated time and cost. Use ISO 8601 duration format (PT1H = 1 hour, PT30M = 30 minutes).

        

        

          
2

          
Add Tools & Supplies

          
List any tools and materials needed. These appear in the rich result and help users prepare.

        

        

          
3

          
Define Each Step

          
Add step-by-step instructions with names, descriptions, and optional images for each step.

        

        

          
4

          
Copy & Implement

          
Copy the JSON-LD and add it to the page that contains the matching how-to content.
