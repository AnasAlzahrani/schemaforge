How to Use Recipe Schema Markup

      

        

          
1

          
Enter Recipe Info

          
Fill in the recipe name, author, description, and a high-quality image URL. Google uses these prominently.

        

        

          
2

          
Add Timing

          
Set prep, cook, and total times using ISO 8601 duration format (PT1H = 1 hour, PT30M = 30 minutes).

        

        

          
3

          
List Ingredients & Steps

          
Add each ingredient on its own line and write clear, step-by-step instructions.

        

        

          
4

          
Add Nutrition & Test

          
Optionally add calorie info, then copy the schema and validate with Google's Rich Results Test.
